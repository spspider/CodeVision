    ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

[!] Важное замечание
[*] Изменения
[-] Исправлена ошибка
[+] Нововведение


v1.05
~~~~~
[*] Разрешено задание ключа через символ '-'.
[+] Сообщения об ошибках.
[+] Файлы описания памяти ввода/вывода AVR процессоров (папка Inc).
[+] Ключ /? - встроенная помощь.
    Ключ /D - имя файла описания I/O (путь указывать разрешено).
    Необходим для замены численного представления псевдонимом в командах:
    cbi, in, out, sbi, sbic, sbis.

v1.04
~~~~~
[-] Поправлены исходники для корректной компиляции subj под Win32.
[+] Введена поддержка регистронезависимых ключей (необязательны).
    Ключ /L - максимальный размер программной памяти процессора.
    Необходим для коррекции относительных переходов, в случае выхода их за
    допустимый диапазон.

v1.02
~~~~~
[*] Поправлен краткий справочник по командам (avrcmd.txt).
[+] Распознование новых команд AVR процессоров (parser.h).

v1.01
~~~~~
[+] Встроенная помощь при запуске subj без параметров.
[+] Новый необязательный параметр [lstfile] - имя создаваемого файла листинга.
    Eсли отсутствует, то получается из имени binfile заменой его расширения на
    lst.
